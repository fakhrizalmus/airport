# Susi Air Pilot Dashboard

Frontend Nuxt untuk pilot dashboard Susi Air. Aplikasi menampilkan login, home dashboard, jadwal bulanan, ringkasan flight-hour limit, status dokumen, dan PWA basics dengan data mock lokal.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev -- -o
```

## Build

```bash
npm run build
npm run preview
```

## Live Deployment

Deployment URL: https://airport-blush.vercel.app/

## Notes

Library yang dipakai:

- Nuxt untuk routing, auto imports, build, dan app structure.
- Tailwind CSS untuk styling mobile-first dan responsive utility classes.
- Pinia untuk state dashboard, schedule, dokumen, dan flight hours.
- Lucide Vue untuk ikon UI yang konsisten dan ringan.
